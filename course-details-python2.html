Intermediate Python     

Python is a programming language which is known              
for its increased productivity, readability, and simplicity.             

Objectives

This course will allow students to continue exploring the Python programming language. It will finish covering the foundational Python topics that students began learning in the Intro to Python course. Then, students will be introduced to applications of Python by learning and developing programs with Pygame and websites with Flask. Students will develop their coding skills through live classes, projects, and presentations. This course is the second of the Python course series. Taking our Intro to Python course prior to this course is recommended but not required.

Eligibility

Intermediate Python will be taught to children in grades 4-8. Students should have a solid understanding of the topics covered in Introduction to Python.
